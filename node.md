## NVM

### Install Node.js with NVM

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
npm install --lts --latest-npm
```

### List versions
```
nvm ls
```

### Use specific version
```
nvm use <version>
```
